# Samagra UP Secondary — NGO Partner Portal

Interactive web portal for NGO partner reporting under Samagra UP Secondary Education Programme, Madhyamik Shiksha Vibhag, Uttar Pradesh.

## Features
- Sign In / Register for NGO partners
- NGO profile setup (one time)
- Monthly KPI report submission with photo upload
- Interactive UP map with NGO markers
- Admin dashboard with report generation
- Print / PDF report for IAS review

## Deploy on GitHub Pages

1. Create a new GitHub repository named `samagra-ngo-portal`
2. Upload `index.html` to the repository
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)`
5. Click **Save**
6. Your portal will be live at: `https://yourusername.github.io/samagra-ngo-portal`

## File Structure
```
index.html    ← Complete portal (single file)
README.md     ← This file
```

## How it works
- Single `index.html` — no server, no database needed
- NGO data updates live within the session
- For permanent data storage, integrate Google Sheets API (next phase)
